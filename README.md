# FABCON2026

This is the supporting repository for code used in FABCON 2026 Fabric Sprawl Session in Atlanta, Ga.

### Repo Folder Information:

- The [FUAM Workspace](/FUAM/) used in demos is in the `FUAM` folder.
- Within the `FUAM` workspace folder is the [_Extensions](/FUAM/_Extensions/) folder where all extensions shown in demos is stored.
- The `DevOps` folder contains the [Semantic Model Pipeline CICD Example](/DevOps/semantic-model-pipelines/).
- All KQL queries shown in the monitoring section of the session live in the [KQL Queries](/KQL%20Queries/) folder.
